### Description

A simple portfolio built with React. 


### Features

- Fully Responsive
- Multi-Page Layout
- Contact Form With EmailJs
- React-Bootstrap

### Setup

Get the code

<pre>git clone https://github.com/drewjordan414/main-portfolio.git</pre>
 
Install required dependencies

<pre>yarn install</pre>


Start the server

<pre>yarn start</pre>
